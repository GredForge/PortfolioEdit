Job Search Application using Firebase Server and Indeed and Meetup API's

This application allows the user to search indeed for job postings and will simultaneously find meetups in the area associated with the searches. The user can then "like" jobs that interested them and have them saved to their user account with the help of google firebase. The technologies used in this app were HTML, CSS, as well as Gooogle Firebase. We used ajax calls to both the Indeed and Meetups Api's.
