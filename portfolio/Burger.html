Burger

Simple full stack app using handlebars for a view engine and Node and MySQL for backend.

This Application uses handlebars to render out our HTML and sequelize to talk to our MySQL database. The application allows you to create burgers for consumption, display those burgers, and add them to the database. You can then display update the database by 'eating' the burger which will change the status on the database from not eaten to eaten and move the burger to the associated div on our page.
